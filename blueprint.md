
# Project Blueprint

## Overview

A Flutter application that provides a web view of the website "https://my.ev.lk". The application is designed to be a simple, fullscreen web browser for the specified URL, with no additional UI elements like headers or navigation buttons. It includes necessary permissions for web browsing, location access, and notifications, and is set up with a custom icon for a professional look.

## Style, Design, and Features

*   **UI:** A minimal, fullscreen WebView.
*   **Color Scheme:** The app will adopt the color scheme of the website it displays.
*   **Typography:** All typography will be inherited from the website.
*   **Iconography:** A custom app icon is used, based on the "ev.lk" logo.
*   **Features:**
    *   Displays the website "https://my.ev.lk".
    *   Requests permissions for Internet, Camera, Location, and Notifications.
